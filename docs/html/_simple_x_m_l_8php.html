◆ simplexml_load_file()

simplexml_load_file (   $filename,
  $class_name = "SimpleXMLElement",
  $options = 0,
  $ns = "",
  $is_prefix = false 
)

Interprets an XML file into an object string $filename Path to the XML file

Libxml 2 unescapes the URI, so if you want to pass e.g. b&#38;#38;c as the URI parameter a, you have to call simplexml_load_file(rawurlencode('https://example.com/?a=' . urlencode('b&#38;#38;c'))). Since PHP 5.1.0 you don't need to do this because PHP will do it for you.

Parameters
string$class_name[optional]

You may use this optional parameter so that simplexml_load_file will return an object of the specified class. That class should extend the SimpleXMLElement class.

Parameters
int$options[optional]

Since PHP 5.1.0 and Libxml 2.6.0, you may also use the options parameter to specify additional Libxml parameters.

Parameters
string$ns[optional]

Namespace prefix or URI.

Parameters
bool$is_prefix[optional]

TRUE if ns is a prefix, FALSE if it's a URI; defaults to FALSE.

Returns
SimpleXMLElement an object of class SimpleXMLElement with properties containing the data held within the XML document, or FALSE on failure.
Since
5.0
