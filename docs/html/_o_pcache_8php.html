(PHP 5 >= 5.6, PECL ZendOpcache >= 7.0.4 )
This function checks if a PHP script has been cached in OPCache. This can be used to more easily detect the "warming" of the cache for a particular script. string $file The path to the PHP script to be checked. bool Returns TRUE if file is cached in OPCache, FALSE otherwise.
