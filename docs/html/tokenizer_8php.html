◆ token_get_all()

token_get_all (   $source,
  $flags = 0 
)

Split given source into PHP tokens string $source The PHP source to parse.

Parameters
int$flags

Valid flags:

  • TOKEN_PARSE - Recognises the ability to use reserved words in specific contexts.

Returns
array An array of token identifiers. Each individual token identifier is either a single character (i.e.: ;, ., >, !, etc...), or a three element array containing the token index in element 0, the string content of the original token in element 1 and the line number in element 2.
