◆ PS_UNRESERVE_PREFIX___halt_compiler()

PS_UNRESERVE_PREFIX___halt_compiler ( )

(PHP 5.1)
Halts the execution of the compiler. This can be useful to embed data in PHP scripts, like the installation files. Byte position of the data start can be determined by the COMPILER_HALT_OFFSET constant which is defined only if there is a __halt_compiler() presented in the file.

Note: __halt_compiler() can only be used from the outermost scope. void
