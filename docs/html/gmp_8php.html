◆ gmp_prob_prime()

gmp_prob_prime (   $a,
  $reps = 10 
)

Check if number is "probably prime" resource|string|GMP $a The number being checked as a prime.

It can be either a GMP number resource, or a numeric string given that it is possible to convert the latter to a number.

Parameters
int$reps[optional]

Reasonable values of reps vary from 5 to 10 (default being 10); a higher value lowers the probability for a non-prime to pass as a "probable" prime.

It can be either a GMP number resource, or a numeric string given that it is possible to convert the latter to a number.

Returns
int If this function returns 0, a is definitely not prime. If it returns 1, then a is "probably" prime. If it returns 2, then a is surely prime.
Since
4.0.4
