  • 'rings' Type: array of GEOSGeometry Rings that can be formed by the costituent linework of geometry.
  • 'cut_edges' (optional) Type: array of GEOSGeometry Edges which are connected at both ends but which do not form part of polygon.
  • 'dangles' Type: array of GEOSGeometry Edges which have one or both ends which are not incident on another edge endpoint
