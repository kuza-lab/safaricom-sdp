memcache_connect (   $host,
  $port,
  $timeout = 1 
)

(PECL memcache >= 0.2.0) Memcache::connect — Open memcached server connection string $host Point to the host where memcached is listening for connections. This parameter may also specify other transports like unix:///path/to/memcached.sock to use UNIX domain sockets, in this case port must also be set to 0.

Parameters
int$port[optional]

Point to the port where memcached is listening for connections. Set this parameter to 0 when using UNIX domain sockets. Note: port defaults to memcache.default_port if not specified. For this reason it is wise to specify the port explicitly in this method call.

Parameters
int$timeout[optional]

Value in seconds which will be used for connecting to the daemon.

Returns
bool Returns TRUE on success or FALSE on failure.
