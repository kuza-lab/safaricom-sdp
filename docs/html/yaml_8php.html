Linebreak types for yaml_emit() YAML_ANY_BREAK Let emitter choose linebreak character. YAML_CR_BREAK Use as break character (Mac style). YAML_LN_BREAK Use as break character (Unix style). YAML_CRLN_BREAK Use as break character (DOS style).
