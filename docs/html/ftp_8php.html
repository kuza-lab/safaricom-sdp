This parameter specifies an alternate port to connect to. If it is omitted or set to zero, then the default FTP port, 21, will be used.

◆ ftp_set_option()

ftp_set_option (   $ftp_stream,
  $option,
  $value 
)

Set miscellaneous runtime FTP options resource $ftp_stream The link identifier of the FTP connection.

Parameters
int$option

Currently, the following options are supported:

runtime FTP options

FTP_TIMEOUT_SEC

Changes the timeout in seconds used for all network related functions. value must be an integer that is greater than 0. The default timeout is 90 seconds.

FTP_AUTOSEEK

When enabled, GET or PUT requests with a resumepos or startpos parameter will first seek to the requested position within the file. This is enabled by default.

Parameters
mixed$value

This parameter depends on which option is chosen to be altered.

Returns
bool TRUE if the option could be set; FALSE if not. A warning message will be thrown if the option is not supported or the passed value doesn't match the expected value for the given option.
Since
4.2
