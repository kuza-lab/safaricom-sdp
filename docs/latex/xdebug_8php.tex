{\footnotesize\ttfamily xdebug\+\_\+debug\+\_\+zval (\begin{DoxyParamCaption}\item[{}]{\$var\+Name }\end{DoxyParamCaption})}

This function displays structured information about one or more variables that includes its type, value and refcount information. Arrays are explored recursively with values. This function is implemented differently from P\+HP\textquotesingle{}s \mbox{\hyperlink{standard__4_8php_a99130912e243988d8395429fd98a4a73}{debug\+\_\+zval\+\_\+dump()}} function in order to work around the problems that that function has because the variable itself is actually passed to the function. Xdebug\textquotesingle{}s version is better as it uses the variable name to lookup the variable in the internal symbol table and accesses all the properties directly without having to deal with actually passing a variable to a function. The result is that the information that this function returns is much more accurate than P\+HP\textquotesingle{}s own function for showing zval information.
