 Obtains an exclusive lock on a given key. 

The execution of the current script will be blocked until the lock can be obtained. Once the lock is obtained, the other scripts that try to request the lock by using the same key will be blocked, until the current script releases the lock by using \mbox{\hyperlink{wincache_8php_aebf492e291ddefd8e13cf61e6e7bbeb1}{wincache\+\_\+unlock()}}.

Local locks are scoped to the application pool in I\+IS Fast\+C\+GI case or to all php processes that have the same parent process identifier.
