The optional {\itshape result\+\_\+type} parameter accepts a constant and determines how the returned array will be indexed. Using {\bfseries{S\+Q\+L\+I\+T\+E\+\_\+\+A\+S\+S\+OC}} will return only associative indices (named fields) while {\bfseries{S\+Q\+L\+I\+T\+E\+\_\+\+N\+UM}} will return only numerical indices (ordinal field numbers). {\bfseries{S\+Q\+L\+I\+T\+E\+\_\+\+B\+O\+TH}} will return both associative and numerical indices. {\bfseries{S\+Q\+L\+I\+T\+E\+\_\+\+B\+O\+TH}} is the default for this function.
