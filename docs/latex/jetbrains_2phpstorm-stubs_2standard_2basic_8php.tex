At startup, P\+HP tries to enable the V\+T100 feature of the S\+T\+D\+O\+U\+T/\+S\+T\+D\+E\+RR streams. By the way, if those streams are redirected to a file, the V\+T100 features may not be enabled.

On Windows these sequences are called Console Virtual Terminal Sequences. \mbox{\hyperlink{patch2_8txt_a3f9992ab8fa942ed02faa63682c3a84c}{resource \$stream  bool \$enable \mbox{[}optional\mbox{]}}}

If {\itshape enable} is specified, the function will try to enable or disable the V\+T100 features of the stream stream. If the feature has been successfully enabled (or disabled), the function will return T\+R\+UE, or F\+A\+L\+SE otherwise.
