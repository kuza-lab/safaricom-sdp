If matches are found for parenthesized substrings of {\itshape pattern} and the function is called with the third argument {\itshape regs}, the matches will be stored in the elements of the array {\itshape regs}. 

\$regs\mbox{[}1\mbox{]} will contain the substring which starts at the first left parenthesis; \$regs\mbox{[}2\mbox{]} will contain the substring starting at the second, and so on. \$regs\mbox{[}0\mbox{]} will contain a copy of the complete string matched.
